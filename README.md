**NYC SCHOOL OF DATA**

This is a repo where you can find downloadable data, tools, and resources related to [Esri's](https://www.esri.com/) presentation at the [2026 NYC School of Data](https://schoolofdata.nyc/) event on Saturday, March 28rd, 2026.

Sign up for a FREE license of ArcGIS Location Platform at the ArcGIS for Developers website:

[https://location.arcgis.com](https://location.arcgis.com)
